# Shagya — Impeccable Design Context

## Design Context

### Users

Indian women shopping for premium sarees online. Mobile-first browsing, visually-driven decisions. Shopping for weddings, festivals, daily wear, and gifts. Context: casual browsing (Instagram → site), intent-driven (searching for specific silk/weave), gift shopping. Emotionally invested — a saree purchase is personal and cultural, not transactional.

### Brand Personality

**Handcrafted, warm, refined.** Shagya sits at the intersection of Indian textile tradition and modern D2C convenience. The interface should feel like walking into a well-curated boutique — personal, warm, never sterile. Every detail should whisper "this was made with care."

### Aesthetic Direction

- **Tone**: Luxe editorial meets warm artisanal. Think museum-quality product photography with clean, breathing layouts. Not maximalist Indian wedding aesthetic — more Sabyasachi minimalism.
- **Theme**: Light mode primary (saree shopping happens during the day, colours must render accurately). Dark mode optional for later.
- **Reference**: Aesop (spacing, restraint), Raw Mango (Indian textile warmth), Toast (craftsmanship storytelling)
- **Anti-references**: Amazon/Flipkart (crowded grids, aggressive CTAs, no breathing room). Myntra/Nykaa Fashion (overdesigned, cluttered). Generic Shopify dropshipping aesthetics.

### Design Principles

1. **Let the sarees be the hero.** Product images carry the weight. Everything else recedes.
2. **Breathe.** Generous white space. One clear action per screen. No competing CTAs.
3. **Warmth over sterility.** Tinted neutrals, rounded details, human-scale typography.
4. **Traditional craft, modern interface.** Old-world quality cues (serif display, craft details) delivered through a fast, modern web experience.
5. **Mobile-first but never mobile-only.** The experience should feel intentionally designed for each breakpoint.

### Tech Stack

Next.js 16, Payload CMS 3.x, Tailwind CSS v4, shadcn/ui, Cloudflare R2, Neon Postgres. Design tokens via CSS custom properties with OKLCH color space.
